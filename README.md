# table-tennis-serve-simulator
Serve Simulator PRO — это интерактивная 3D-видеолаборатория и тактический симулятор для глубокого анализа и отработки профессиональных подач в настольном теннисе. Приложение работает на базе откалиброванного физического движка и визуализирует баллистику полета мяча в реальном времени прямо в браузере.
