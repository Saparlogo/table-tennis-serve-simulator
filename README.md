# 🏓 Serve Simulator PRO v3.0

**Serve Simulator PRO** — это передовая интерактивная 3D/2D-видеолаборатория и тактический тренажер баллистики для детального анализа и отработки профессиональных подач в настольном теннисе. Приложение позволяет кастомизировать параметры удара, симулировать полет мяча с учетом реальной аэродинамики и изучать траекторию на премиальном оборудовании уровня мировых первенств.

Проект спроектирован в гибкой гибридной архитектуре: он может работать как автономная Single Page система внутри одного оптимизированного файла `index.html`, так и разворачиваться в виде модульного масштабируемого React/Vite/TypeScript веб-приложения.

## 🚀 Ключевые особенности и функционал

### 1. Физический движок баллистики
Расчет полета мяча основан на численном интегрировании метода Эйлера с шагом дискретизации dt = 0.001с. Движок учитывает:
* **Гравитацию и силу лобового сопротивления воздуха (Drag Force):** Сила сопротивления воздуха рассчитывается динамически в зависимости от текущей квадратичной скорости, площади сечения мяча и плотности воздушной среды.
* **Калиброванный эффект Магнуса (Magnus Force):** Вращающийся легкий целлулоидный мяч весом 2.7 грамма испытывает поперечное ускорение до 12 м/с², искривляющее траекторию в полете. Сила Магнуса вычисляется через векторное произведение угловой и линейной скоростей: F_m = M_f × (ω × v).
* **Физику соударения с фрикционным kick-эффектом:** При касании стола вертикальная скорость отражается с коэффициентом восстановления, а угловое вращение (Spin) за счет силы трения преобразуется в мгновенное линейное ускорение. Реализована физика «Призрачной подачи» (Ghost Serve) — экстремальное нижнее вращение заставляет мяч покатиться обратно к сетке после отскока.

### 2. Управление интерактивной 3D-камерой (Orbit & Zoom)
Центральный 3D-экран симуляции полностью интерактивен:
* **Вращение ракурса (ЛКМ):** Зажмите левую кнопку мыши и двигайте её в любом направлении, чтобы облететь вокруг стола на 360° и рассмотреть подачу с позиции принимающего, судьи или от сетки.
* **Масштабирование (Колесико мыши):** Прокручивайте колесико мыши для плавного Zoom In / Zoom Out, чтобы приблизиться к ракетке в момент контакта или детально изучить пролет мяча над сеткой.

### 3. Тактическая 2D-доска (Hawk-Eye Minimap)
В верхний левый угол интегрирован дополнительный контрастный планшет. Он прорисовывает точную проекцию траектории на плоскость стола (вид строго сверху), фиксируя планиметрический занос мяча, синюю линию сетки и маркеры-крестики ✕ в точках соударения с цветовой кодировкой (оранжевый — 1-й отскок, зелёный — 2-й).

## 🎬 Структура 10-секундного интерактивного видеоролика

Каждая симуляция рендерится в формате полноценной 10-секундной шкалы времени (таймлайна) с возможностью перемотки, паузы и покадрового анализа:

| Время | Фаза | Описание |
|---|---|---|
| 0.0s – 2.0s | 🟢 Концентрация | Мяч неподвижно лежит на открытой ладони виртуального игрока |
| 2.0s – 4.0s | 🟡 Подброс | Мяч взлетает вертикально по параболе (правило ITTF — минимум 16 см) |
| 4.0s | 💥 Контакт | Анимация резкого кистевого удара, передающего скорость и RPM вращения |
| 4.0s – 6.0s | 🔵 Полет | Плавный полет в 60 FPS по баллистической траектории с валидацией правил |
| 6.0s – 10.0s | 🟣 Slow-Mo Replay | Hawk-Eye разбор x0.25: неоновая кривая + векторы физических сил |

## 🏓 Премиальный дизайн оборудования (Butterfly Style)

* **Стол Butterfly Centrefold 25:** Игровая поверхность имеет матовый антибликовый цвет *Butterfly Blue* (`#1c355e`). Отрисована объемная 3D-рама отбортовки столешницы, прочные стальные ножки-опоры, уходящие в пол, и выносные стойки крепления сетки, выступающие на регламентированные **15.25 см** за боковую кромку стола.
* **Ракетка Butterfly Viscaria:** Виртуальная ракетка получила фирменную сине-белую продольную полоску на ручке основания *Viscaria*, матовые профессиональные накладки (красная *Tenergy* / черная *Dignics*) и тонкую ярко-оранжевую торцевую ленту, имитирующую знаменитую технологичную губку *Spring Sponge*.

## ⚙️ Профессиональная библиотека подач

Симулятор содержит 5 встроенных пресетов профессионального уровня с возможностью индивидуальной кастомизации высоты подброса, точки удара, резкости кисти и коррекции скорости:

1. **⏱ Маятник (Pendulum)** — Сильное боковое правое + нижнее вращение уводит мяч по крутой дуге в сторону.
2. **🔄 Обратный маятник (Reverse Pendulum)** — Зеркальное левое вращение, мяч уходит в противоположный от ожидаемого угол.
3. **🪓 Томагавк (Tomahawk)** — Агрессивное верхне-боковое вращение. Мяч резко пикирует в стол и выстреливает в сторону после отскока.
4. **👻 Призрачная подача (Ghost Serve)** — Экстремальный бек-спин. Мяч притормаживает после отскока и катится назад к сетке.
5. **🚀 Высокий подброс (High Toss)** — Подброс до 3 м аккумулирует гравитационную скорость, усложняя чтение вращения.

## 🛠 Локальный запуск и развертывание

### Способ 1 — Автономный запуск без установки зависимостей

Запустите локальный сервер в папке проекта через Python:
```bash
python -m http.server 3000 --bind 127.0.0.1
```
Затем откройте в браузере: **http://127.0.0.1:3000**

> Приложение полностью функционально без какой-либо установки пакетов.
> Tailwind CSS и шрифты Google Fonts (Inter, JetBrains Mono) подключены через официальные CDN-ссылки.

### Способ 2 — Сборка через Node.js / Vite

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера с горячей перезагрузкой (HMR)
npm run dev

# Production-сборка в папку /dist
npm run build
```
